PBS Tools
by Sam Wade

Tools
Efficently generate and output values to be used in PBS data.

Item Codes
PBS

Gets all the PBS item codes associated with a given drug name or ATC code.
Active Authority Codes
PBS

Uses the public PBS API to identify the active authority codes for a given drug name or ATC code.
Historical Authority Codes
PBS | Restricted Use

Uses combinations mapped from PBS claims files to identify active and historical authority codes.
Defined Daily Dose
PBS

Calculates and visualises the WHO Defined Daily Dose for the item codes associated with a given drug name or ATC code.
Under Co-Payment Visualiser
PBS

Visualises if given item codes fall over or under the co-payment threshold for years of PBS claims that don't record under co-payment dispensings.

Map Generators
Code to generate the mapping files used for the tools on this site.

Item Codes with Historical Flag
PBS

Uses the PBS API to add a historical flag variable for each item code.
Active Authority Code Map
PBS

Uses the PBS API to map item codes to their restriction IDs, authority codes, and restriction texts for the latest schedule.
Defined Daily Dose Map
PBS

Uses the PBS API and data extracted from the WHO website to estimate the defined daily dose for each item code.
